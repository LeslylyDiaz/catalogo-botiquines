# Comandos básicos de Git

## git --version
Muestra la versión instalada de Git.

Ejemplo:
```bash
git --version
```

---

## git init
Inicializa un repositorio Git en la carpeta actual.

Ejemplo:
```bash
git init
```

---

## git add
Agrega archivos al área de preparación (staging).

Ejemplo:
```bash
git add .
```

---

## git commit
Guarda los cambios realizados con un mensaje descriptivo.

Ejemplo:
```bash
git commit -m "Primer commit"
```

---

## git status
Muestra el estado actual del repositorio.

Ejemplo:
```bash
git status
```
